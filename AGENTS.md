# Editar contenido

**Todo el contenido editable está en `content/landing.ts`.**

Si necesitás cambiar textos, títulos, descripciones, testimonios, estadísticas, links o imágenes, ese es el único archivo que tenés que tocar. No hay que modificar ningún componente.

```
content/
└── landing.ts
```

Las imágenes van en `public/`. Se referencian desde `landing.ts` como `/nombre-archivo.png`.
